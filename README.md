# RPN-Calculator
Reverse Polish Notation Calculator

Directions

1. Enter first number (number to be operated on)
2. Enter second number (number to operate with)
3. Enter operand:
              + add
              - subtract
              * multiply 
              / divide
              n negate top item
              d duplicate top item
              r reverse top two items
              p print top item
              c clear entire stack
              a print all items
              q quit
              
Example:

input:
1. 5
2. 2
3. -

equivalent to 5 - 2

output:
3
